# On_the_Way_To_the_Gym

請在 root directory 新增 **.env file**，並加入以下環境變數:

```
1.  DATABASE
2.  DATABASE_USER_NAME
3.  DATABASE_PASSWORD
4.  DATABASE_HOST
5.  DATABASE_PORT
6.  DATABASE_DIALECT
7.  CHANNEL_ID
8.  CHANNEL_SECRET
9.  CHANNEL_ACCESS_TOKEN
```

然後,
1.  執行 `npm install`
2.  執行 `npm i -g nodemon`
2.  執行 `nodemon | node app.js`
